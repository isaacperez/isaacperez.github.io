Projects

In this section I have tried to compile the main projects that I have done since the last years of my degree.

Personal projects

Fish segmentation

Pytorch project for fish segmentation (2021)
This project is used for training and testing a convolutional neural network for semantic segmentation of fish in images. The project is built with Python using the PyTorch library.

  • Repository: https://github.com/isaacperez/fish-segmentation

Tensorflow 1.2 model to Raspberry Pi C code

Transforming a Tensorflow 1.2 model to Raspberry Pi C code (2018)
A simple way to get a simple C code that runs a TensorFlow model on Raspberry Pi. The project is built with Python.

  • Repository: https://github.com/isaacperez/tf12raspberry

Embedded solution

NVIDIA Jetson TX2 + IntelRealSense Depth camera + ROS + OpenCV + VisionWorks (2018)
This project allows the creation of a solution for ROS using the OpenCV and VisionWorks libraries. In addition, it is possible to read images from IntelRealSense cameras. The project is developed in C++ for NVIDIA Jetson TX2.

  • Repository: https://github.com/isaacperez/embedded-solution

Professional projects

Tool for pathology recognition

Tool for pathology recognition in capsule endoscopic images using deep learning (2021) During my time at the University, we carried out a project with the Juan Ramón Jimenez Hospital to create a tool for help the specialist to detect certain pathologies in capsule endoscopic images using a deep learning model.

  • Link: https://doi.org/10.1007/s10489-022-04146-3

Automated biodiversity monitoring system based on deep learning and citizen science

Automated biodiversity monitoring system in the Iberian Peninsula (2020)
During my time at the University, we developed a deep learning system for classifying camera trap images. We created a project on Zooniverse to use citizen science. Our project has been one of the most active on the platform.

  • Zoonvierse project: https://www.zooniverse.org/projects/aicensusuhu/iberian-camera-trap-project

  • Press release: https://theconversation.com/la-inteligencia-artificial-nos-ayuda-a-estudiar-la-fauna-de-donana-para-mejorar-su-conservacion-162815+

EyeVIEW

System for the detection of printing errors on metal foils (2018)
In this project I designed from scratch (including the choice of cameras and lighting) a computer vision solution with the aim of inspecting prints on metal foils. The system was developed entirely in C++ for the main manufacturer of closures in the south of Spain, Torrent Closures.

Strawberry instance segmentation

Strawberry instance segmentation at Agrobot (2017)
During my time at Agrobot I designed the strawberry instance segmentation algorithm used by the harvesting robot. At that time, there were no models like Mask R-CNN. NVIDIA made a video about my algorithm in its I AM AI Docuseries.

  • NVIDIA programme: https://www.youtube.com/watch?v=bXQg_M7_6_E

Academic projects

Blood vessel segmentation

A new deep learning method for blood vessel segmentation in retinal images based on convolutional kernels and modified U-Net model (2021)
In this article we managed to modify the U-Net network and introduce new training techniques that allowed us to make the proposed model the best performing of the state of the art.

  • Link: https://doi.org/10.1016/j.cmpb.2021.106081

Strawberry instance segmentation - Round 2

A new deep-learning strawberry instance segmentation methodology based on a fully convolutional neural network (2021)
In this paper we introduce a new instance segmentation paradigm that allows using fully convolutional neural networks to do instance segmentation. Thanks to this new paradigm, the model achieves 6 times more fps than Mask R-CNN and 15% better results.

  • Link: https://doi.org/10.1007/s00521-021-06131-2

DEEP LEARNING

DEEP LEARNING: Fundamentals, Theory and Application (2021)
Taking advantage of the fact that part of my teaching was dedicated to deep learning, I wrote this book to try to help students understand the most elementary concepts behind neural networks, recurrent neural networks and convolutional neural networks. It is accompanied by detailed illustrations that can help you understand many of the concepts. The book is available free of charge.

  • Book: http://www.uhu.es/publicaciones/?q=libros&code=1252

Strawberry instance segmentation - Round 1

A fast and accurate deep learning method for strawberry instance segmentation (2020)
In this article we introduced a series of modifications to Mask R-CNN that allowed us to double the fps without an appreciable loss in accuracy. The use of such models in real-world environments requires optimised models for performance-limited environments.

  • Link: https://doi.org/10.1016/j.compag.2020.105736

StrawDI

The Strawberry Digital Images Data Set (2020)
The first public data set of strawberry images for instance segmentation. This dataset contains 8000 images of strawberries. From the total set, 3100 images have been selected at random. For each of these images a ground truth labelled image has been manually generated containing the individual segmentation of all the strawberries appearing in the image.

  • Download: https://strawdi.github.io/

Student projects

Master’s Degree Final Project

Strawberries instance segmentation for an automated strawberry harvester (2020)
In this project, the individual segmentation of each strawberry is obtained by means of instance segmentation. For this purpose, Mask R-CNN was implemented from scratch from the original paper. To train the model, images were collected in strawberry plantations and individual strawberries were marked.

Bachelor’s Degree Final Project

Detection, segmentation and classification of strawberries for an automated strawberry harvester (2017)
For my Bachelor’s Degree Final Project I designed a system to individually segment the strawberries present in an image to determine their position and degree of maturity. The system is composed of three convolutional neural networks that detect, segment and classify the strawberries based on their ripeness. The choice of components and computer configuration for training the models was part of this work.

The Mario AI Benchmark

Intelligent agent to play The Mario AI competition (2017)
The Mario AI competition is about creating an agent capable of playing the Super Mario game. For this project I programmed an agent in JAVA that was trained by reinforcement learning with the Q-learning algorithm.

  • Demo: https://www.youtube.com/watch?v=j1IVecHPL-Q

  • Repository: https://github.com/isaacperez/universidad/tree/master/SuperMario

Rule-based autonomous driver compiler for Torcs

Rules-based system for playing Torcs with a compiler that generates the player (2016)
The aim of this project is to build a compiler that generates from a set of rules an autonomous driver for Torcs. A JAVA code capable of playing Torcs is generated by means of a rules language that we invented. The rule system is based on the use of fuzzy logic.

  • Repository: https://github.com/isaacperez/universidad/tree/master/CompiladorControladorTorcs
